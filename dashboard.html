📖 AI会社の使い方

1

AI会社を起動する

コマンドプロンプト（cmd）を開いて以下を実行：

# まず移動
cd C:\Users\masat\OneDrive\Desktop\company

# APIキーを設定（毎回必要）
set ANTHROPIC_API_KEY=sk-ant-ここにキーを貼る

# 起動
python main.py

起動すると 👑 オーナー指示 > と表示されます。

2

指示の出し方（例文集）

自然な日本語で話しかけるだけでOKです：

👑 オーナー指示 > オーストラリア留学ガイドを作って。
費用・都市比較・語学学校・ビザの内容で、
canada.htmlと同じ形式のHTMLにして

👑 オーナー指示 > A8.netで提携申請するための
自己紹介メールのテンプレートを作って

👑 オーナー指示 > Instagramに投稿する「カナダ留学費用比較」の
キャプションと画像案を作って

👑 オーナー指示 > 競合サイトを調べて、うちのサイトが
勝てるポイントを分析して
3

CEOが自動で従業員に振り分ける

指示を出すと、AI CEOが内容を分析して最適な従業員（researcher・writer・strategistなど）に 自動でタスクを振り分けます。
あなたは待つだけで、数分後にレポートが完成します。

4

成果物をサイトに反映する

CEOのレポートに記事が出てきたら、
ryugaku-direct/articles/ フォルダに .html ファイルとして保存。
その後 Netlify に再アップロードすれば公開完了です。

💡 週ごとの使い方のコツ

  • 月曜：今週やることをCEOに宣言する（「今週はNZの記事を作りたい」）
  • 平日：CEOの成果物をチェック、Instagramに投稿する
  • 週末：新しい記事をサイトにアップ、来週の方針を決める
